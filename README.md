# Consumer Web App

> Consumer Web App built in nextJS (demo -> https://n81z0.sse.codesandbox.io/)

### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd Consumer-Web-App
$ yarn install
$ yarn run dev
```

For production environments...

```sh
$ cd Consumer-Web-App
$ yarn install
$ yarn run build
```
